# To-Do List Application

A simple Python To-Do List application that allows users to manage their tasks. Tasks are saved to a file for persistent storage between sessions.

## Features
- View tasks
- Add new tasks
- Delete tasks
- Save tasks to a file (`tasks.txt`)
